THE THESIS

GPT-3.5 arrived. Then GitHub Copilot. Then Claude, Cursor, Codex. Each generation doubled capability and halved cost. Today, AI writes production code, debugs systems, and ships features autonomously. Jensen Huang said it plainly at CES 2025: "The marginal cost of software is going to zero."

History says this won't shrink the software industry — it will explode it. When steam engines became more efficient, coal consumption rose. When spreadsheets made calculation free, firms hired more accountants, not fewer. The car didn't contract the auto industry; it created one worth trillions. When building software costs nothing, the number of applications built trends toward infinity.

The second-order effects are vast. Distribution becomes the new moat. Hyper-niche software — too expensive to justify before — becomes viable overnight. Solo operators run portfolios of products. Agents become buyers. Software fragments and personalizes down to the individual. As Marc Andreessen put it: "The amount of software in the world is about to explode by 1000x."

Limit 0 Labs exists at this frontier. We ship products across finance, design, and beyond — in a world where the only limit is ideas, not cost.

SECOND ORDER EFFECTS
01
JEVONS PARADOX

In 1865, William Jevons observed that more efficient steam engines increased total coal consumption — not decreased it. Cheaper software doesn't mean less software. It means an explosion of applications across every domain, niche, and individual use case imaginable.

02
THE LONG TAIL

When development cost drops from $500K to $500, a product serving 500 users becomes a viable one-person business. Hyper-niche software — fleet management for alpaca farms, compliance tools for small-batch cosmetics — becomes economically rational for the first time.

03
DISTRIBUTION AS MOAT

When anyone can build, those who can reach, retain, and earn trust win. Joel Spolsky's Law of Commoditized Complements applies: building is now the commodity. Brand, network effects, and distribution capture the value.

04
AGENTS AS BUYERS

AI agents are becoming purchasing decision-makers. SEO becomes AEO — Agent Engine Optimization. Software needs to be agent-discoverable: machine-readable specs, programmatic trials, verifiable benchmarks. The rules of distribution are being rewritten from the ground up.
