I. The Thesis
lim (cost) → 0

The limit in calculus describes a value a function approaches. Limit 0 Labs exists at the point where the cost of building software converges on zero.

GPT-3.5 arrived. Then GitHub Copilot. Then Claude, Cursor, Codex. Each generation doubled capability and halved cost. Today, AI writes production code, debugs systems, and ships features autonomously. Jensen Huang said it plainly at CES 2025: "The marginal cost of software is going to zero."

The hottest new programming language is English.

Andrej Karpathy — Co-founder, OpenAI

History says this won't shrink the software industry — it will explode it. When steam engines became more efficient, coal consumption rose. When spreadsheets made calculation free, firms hired more accountants. The car didn't contract the auto industry; it created one worth trillions. When building software costs nothing, the number of applications built trends toward infinity.

The second-order effects are vast. Distribution becomes the new moat. Hyper-niche software — too expensive to justify before — becomes viable overnight. Solo operators run portfolios of products. Agents become buyers. Software fragments and personalizes down to the individual. As Marc Andreessen put it: "The amount of software in the world is about to explode by 1000x."

Limit 0 Labs exists at this frontier — shipping products across finance, design, and beyond — in a world where the only limit is ideas, not cost.

III. Second-Order Effects
Effect 01
Jevons Paradox

In 1865, William Jevons observed that more efficient steam engines increased total coal consumption — not decreased it. Cheaper software doesn't mean less software. It means an explosion of applications across every domain, niche, and individual use case that was previously economically irrational to build.

Effect 02
The Long Tail

When development cost drops from $500K to $500, a product serving 500 users at $10/month becomes a viable one-person business. Hyper-niche software — fleet management for alpaca farms, compliance tools for small-batch cosmetics — becomes economically rational for the first time.

Effect 03
Distribution as Moat

When anyone can build, those who reach, retain, and earn trust win. Joel Spolsky observed in 2002 that commoditizing a complement captures the value above it. Building is now the commodity. Brand, network effects, and distribution capture the value.

Effect 04
Agents as Buyers

AI agents are becoming purchasing decision-makers. SEO becomes AEO — Agent Engine Optimization. Software needs to be agent-discoverable: machine-readable capability descriptions, standardized benchmarks, programmatic trial environments. The rules of distribution are being rewritten.
